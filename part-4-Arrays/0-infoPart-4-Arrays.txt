@Types of Arrays

1. One-dimensional (1D) Arrays or Linear Arrays,
 Example: int marks[10].

2. Multi-dimensional array,
 (a) two-dimensional (2D) Arrays or Matrix Arrays,
  Example: int marks[2][3]. // row column.

 (b) three-dimensional array
   Example: int marks[2][3][2].
